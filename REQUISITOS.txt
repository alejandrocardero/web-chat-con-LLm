================================================================================
                           CHAT LLM - REQUISITOS Y USO
================================================================================

DESCRIPCIÓN
-----------
Chat LLM es una aplicación web local que te permite chatear con modelos de 
lenguaje (LLM) ejecutados localmente mediante Ollama. La aplicación usa 
almacenamiento local (localStorage) para guardar tus conversaciones, por lo 
que todos los datos permanecen en tu navegador.


================================================================================
REQUISITOS PREVIOS
================================================================================

1. NODE.JS Y NPM
   ---------------
   - Node.js versión 18 o superior
   - npm (viene incluido con Node.js)
   
   Descargar desde: https://nodejs.org/
   
   Verificar instalación:
   > node --version
   > npm --version


2. OLLAMA
   ------
   - Ollama instalado y configurado
   - Al menos un modelo descargado
   
   Descargar desde: https://ollama.com/
   
   Después de instalar Ollama:
   
   a) Descargar un modelo (ejemplos):
      > ollama pull glm-4.6:cloud
      > ollama pull llama3.2
      > ollama pull mistral
      > ollama pull phi3
   
   b) Ver modelos instalados:
      > ollama list
   
   c) Iniciar Ollama (generalmente inicia automáticamente):
      > ollama serve


================================================================================
INSTALACIÓN
================================================================================

1. NAVEGA A LA CARPETA DEL PROYECTO
   ---------------------------------
   > cd "C:\Users\PC\nexus chat"


2. INSTALA LAS DEPENDENCIAS
   -------------------------
   > npm install
   
   Esto descargará todos los paquetes necesarios (React, Vite, etc.)


3. INICIA LA APLICACIÓN
   ---------------------
   > npm run dev
   
   La aplicación se abrirá automáticamente en:
   http://localhost:5173


================================================================================
CONFIGURACIÓN INICIAL
================================================================================

1. PRIMERA VEZ QUE USAS LA APP
   ---------------------------
   - La aplicación viene preconfigurada con:
     * URL: http://localhost:11434/v1
     * Modelo: glm-4.6:cloud (o el que tengas instalado)
   
   - Si necesitas cambiar la configuración:
     1. Haz clic en "Configurar Ollama" (abajo en el sidebar)
     2. Ajusta la URL y el nombre del modelo
     3. Haz clic en "Probar conexión" para verificar
     4. Haz clic en "Guardar"


2. VERIFICAR QUE OLLAMA ESTÁ FUNCIONANDO
   --------------------------------------
   Abre una terminal y ejecuta:
   > curl http://localhost:11434/api/tags
   
   Deberías ver una lista de tus modelos instalados.


================================================================================
CÓMO USAR LA APLICACIÓN
================================================================================

1. CREAR NUEVA CONVERSACIÓN
   -------------------------
   - Haz clic en el botón "+" en la parte superior del sidebar
   - Se creará una nueva conversación titulada "Chat 1", "Chat 2", etc.


2. ENVIAR MENSAJES
   ----------------
   - Escribe tu mensaje en el área de texto
   - Presiona ENTER para enviar
   - Presiona SHIFT+ENTER para salto de línea
   
   - También puedes:
     * Adjuntar archivos de audio (botón del micrófono)
     * Adjuntar documentos (botón del clip)


3. GESTIONAR CONVERSACIONES
   -------------------------
   - Renombrar: Haz clic en el lápiz junto al nombre
   - Eliminar: Haz clic en el icono de basura
   - Cambiar entre conversaciones: Haz clic en el nombre


4. CONFIGURAR EL MODELO
   ---------------------
   - Haz clic en "Configurar Ollama" en el sidebar
   - Ajusta los parámetros:
     * Ollama URL: Dirección del servidor Ollama
     * Modelo: Nombre del modelo a usar
     * Temperatura: Creatividad (0 = preciso, 2 = creativo)
     * Máx. tokens: Longitud máxima de respuesta


================================================================================
SOLUCIÓN DE PROBLEMAS
================================================================================

PROBLEMA: La aplicación no carga
--------------------------------
SOLUCIÓN:
  1. Verifica que Node.js esté instalado: > node --version
  2. Ejecuta: > npm install
  3. Ejecuta: > npm run dev


PROBLEMA: No recibo respuesta de la IA
---------------------------------------
SOLUCIÓN:
  1. Verifica que Ollama esté corriendo: > ollama list
  2. Si no hay modelos: > ollama pull glm-4.6:cloud
  3. Abre la consola del navegador (F12) y revisa los errores
  4. Verifica la configuración en "Configurar Ollama"


PROBLEMA: Error de conexión con Ollama
--------------------------------------
SOLUCIÓN:
  1. Inicia Ollama: > ollama serve
  2. Verifica la URL en la configuración (debe ser http://localhost:11434/v1)
  3. Prueba la conexión desde el navegador:
     http://localhost:11434/api/tags


PROBLEMA: El modelo no está disponible
--------------------------------------
SOLUCIÓN:
  1. Lista los modelos instalados: > ollama list
  2. Si no está el modelo, descárgalo: > ollama pull <nombre-modelo>
  3. Actualiza el nombre del modelo en la configuración


PROBLEMA: La pantalla se oscurece al enviar
-------------------------------------------
SOLUCIÓN:
  - Esto es normal mientras se procesa la respuesta
  - Si permanece oscurecida, revisa la consola (F12) para ver errores
  - Puede ser que Ollama esté procesando una respuesta larga


PROBLEMA: Las conversaciones desaparecen al recargar
----------------------------------------------------
SOLUCIÓN:
  - Las conversaciones se guardan en localStorage del navegador
  - No borres los datos de navegación del sitio
  - Usa siempre el mismo navegador

================================================================================
NOTAS IMPORTANTES
================================================================================

1. PRIVACIDAD
   - Todos los datos se guardan LOCALMENTE en tu navegador
   - No se envía información a servidores externos
   - Las conversaciones con la IA se procesan localmente con Ollama


2. RENDIMIENTO
   - La velocidad de respuesta depende del modelo que uses
   - Modelos más grandes = respuestas más lentas pero mejores
   - Recomendado: glm-4.6:cloud, llama3.2, o phi3 para velocidad


3. NAVEGADORES SOPORTADOS
   - Chrome/Edge (recomendado)
   - Firefox
   - Safari


4. MODO OFFLINE
   - La aplicación requiere que Ollama esté corriendo
   - Sin Ollama, la interfaz carga pero no puede chatear
